
# neovim-config
This is my neovim config for Ruby On Rails web development(at least for now).

## Basic keymap
- Leader key: `;`
- Toggle file tree: `<A-m>`
- Global filename search: `<leader>ff`
- Global keyword search: `<leader>fg`
- List buffer: `<leader>fb`
- Emmet complete: `<C-y>,`
- Ace Jump(bottom): `t`
- Ace Jump(top): `T`
